# Ansible Rexray

Ansible role to install/configure [Rexray](https://rexray.readthedocs.io/) uma solução de gerenciamento de armazenamento para oferecer suporte em tempo de execução de contêiner, como Docker. O REX-Ray permite que aplicativos com estado, como bancos de dados, persistam e mantenham seus dados após o término do ciclo de vida do contêiner
